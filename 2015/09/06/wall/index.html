翻过那座墙，你能到达世界任何角落

2015-09-06

最近发现家里的宽带已经无法正常使用 OpenVPN 来看 Youtube 了。卡卡卡卡简直卡的一笔。

不过身为一名中国人其实已经很习惯了各种不顺畅了，近 3 年墙也是迅猛发展，开始尝试各种新技术，加大封锁面了。不知不觉它的存在已经非常影响自己的生活了。

毋庸置疑正是它的存在，国内的互联网企业才可以如此迅猛发展；同时也正因如此，国内的搜索简直就像扶不起的阿斗给出来的结果完全看不下眼。想当年 Google 还能用的时候，碰到地雷连接被重置的那会，被重置了再用 Baidu 恩。

不过最近几年实在过头了，国外 N 多技术的网站都已经打不开了。同时还把 Akamai， Cloudflare， GoogleCDN 这类 CDN 封了，导致一半以上的国外网站打不开，或者打开完全看不了因为静态资源和样式全跪了。对于做技术的人简直就是一场噩梦。你试试用 Baidu 搜搜技术的东西……简直心痛。啊我的心好痛。

最近几年的墙，已经从被动变为主动了。已经成为维护国家稳定的利器，最近 Github 被打的骂娘，但还是扛下来了。约谈各位翻墙工具作者，让其下架。

其实对于那些互联网工作者，他们无非是希望能用上好工具，和外国朋友聊天而已。Google 替代 Baidu 查论文，查技术文档、Instagram 晒照片，用 Facebook 和外国朋友聊天。他们都是不党员，何必担心他们被怂恿退党呢？

翻墙成本开始提高，从当年免费工具大把大把，到现在一个两个全部沦陷。如果希望长久的，估计还是需要自己配置了。曲径当年做的非常好，有 90% 以上的可用率，速度极快，看 youtube 1080p 都不卡，可惜已经没了。

所以还是考虑用 Shadowsocks 或者 OpenVPN 自己搭吧。VPN 还是有很多很棒的选择的，除了 OpenVPN，其实 Shadowsocks 的作者写了一个 ShadowVPN 也特别棒。

舍得投钱就用青云的香港主机，真的快，穷的也没关系，DigitalOcean 一个月才 5 刀够用，但是没那么快而已。如果你觉得一个月 30 块钱都不想花，你还是用 Baidu 吧。人生那么断，不要在浪费时间了。

Shadowsocks 是 SOCK5 代理，OpenVPN 是 VPN。一个应用层一个网络层。其实看自己的需求去用。

Shadowsocks 就不介绍了，网上太多教程了。

如果公司希望有一套翻墙解决方案，那么 VPN 还是最好的解决办法。几个步骤可以让整个公司国内外分流自由通行，最重要的是：任何人只要连上公司的网络就可以用了。

假定公司是用一台 Linux 做为网关服务器，如果你用的是路由器或者防火墙，你需要自己想想办法了。

  • 准备一个好的香港主机，以及一个备用节点（日本或者美国）。
  • 从网上找一下国内的 CIDR 表。转换成路由表。
  • 国内的 CIDR 以及国外主机的 IP 加到国内网关。
  • 其他的默认走 VPN 网关。
  • 写一个脚本去 ping VPN 网关，不可用时自动切换。

当然你也可以不做高可用，但是翻墙断的时候就是整个公司都会骂娘的 -。- 听闻某家公司作死用了全套 Google for Work 办公。连邮箱也是用 Gmail。试想人家在用 Gmail 发邮件发到一半断了会怎样。

不过 Google Doc 的协作真的太好用了！N 个人同时编辑同一个文档，你可以看到人家的光标一点一点在移动！

仅仅是个思路，具体实现还是自己研究吧。哟西最近发现 Youtube 上有好多好棒的技术视频，高清无码，简直无法自拔。就酱，88.
